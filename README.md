# DroneAudioNet — anonymous project page

This repository hosts the **static project webpage** (HTML, CSS, audio examples, and spectrogram figures) for double-blind review. It is **not** the training or inference codebase.

**Training, inference, and model code** are linked below (GitHub for source; Google Drive for checkpoints).

**To download the code:** **[https://github.com/DroneAudioNet/DroneAudioNet-Code](https://github.com/DroneAudioNet/DroneAudioNet-Code)**

**To download the model checkpoints:** **[https://drive.google.com/drive/folders/1l17jAIRBSJjdzdHmjoWUZ8EjwV7oAbQq](https://drive.google.com/drive/folders/1l17jAIRBSJjdzdHmjoWUZ8EjwV7oAbQq)**
